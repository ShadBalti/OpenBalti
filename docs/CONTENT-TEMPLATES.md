### SEO Requirements for Blog Posts

```
✓ Title: 40-60 characters, includes primary keyword
✓ Meta Description: 150-160 characters, compelling, includes keyword
✓ H1: One per page, includes primary keyword
✓ H2/H3: Logical hierarchy, natural keyword placement
✓ Word Count: 1,500-2,000+ words minimum
✓ Internal Links: 3-5 relevant, descriptive anchor text
✓ External Links: 3-5 authoritative sources
✓ Images: 1 per 300-500 words, alt text on all
✓ Featured Image: High quality, 1200x630px minimum
✓ Reading Time: Accurate calculation (180 words = 1 min)
✓ Author Info: Credentials and expertise clear
✓ Date: Creation and update dates included
```

## Writing Guidelines

### Tone & Style

- **Tone:** Warm, educational, respectful, encouraging
- **Audience:** Language learners of all levels
- **Clarity:** Use simple language, explain technical terms
- **Engagement:** Tell stories, use examples, be conversational
- **Accuracy:** Verify all facts, cite sources
- **Inclusivity:** Respect cultural sensitivity, acknowledge diverse perspectives

### Word Choice

**Avoid:**
- Jargon without explanation
- Condescending tone
- Overly technical language
- Cultural stereotypes

**Use:**
- Active voice (80%+ of the time)
- Concrete examples
- Relatable scenarios
- Inclusive language

### Formatting Best Practices

```markdown
✓ Use headings for structure (H2 for sections, H3 for subsections)
✓ Use bullet points for lists (no more than 5-7 items)
✓ Use bold for key terms and concepts
✓ Use italics for emphasis (sparingly)
✓ Use blockquotes for external quotes
✓ Use code blocks for examples or structured data
✓ Use tables for comparisons
✓ Use images to break up text
✓ Use callout boxes for important notes
✓ Keep paragraphs to 3-4 sentences max
✓ Keep sentences under 20 words when possible
```

## Content Expansion Guide

### Expanding Thin Content (Current: 300-500 words → Target: 1,500-2,000 words)

**Step 1: Audit Existing Content**
- Keep all valuable information
- Identify gaps and missing subtopics
- Note where examples could be added
- Mark places for images/media

**Step 2: Add New Sections**
- Introduction (expand to 150-200 words)
- 3-4 major sections (400-600 words each)
- Key takeaways/summary (150-200 words)
- Related resources section (100-150 words)

**Step 3: Enhance Existing Sections**
- Add real-world examples
- Include cultural context
- Add audio/video where relevant
- Improve formatting with headers

**Step 4: Add Learning Resources**
- Links to related learn pages
- Dictionary word suggestions
- Practice exercises
- Further reading

**Step 5: Optimize for SEO**
- Add internal links
- Improve meta description
- Ensure proper heading structure
- Add schema markup

## Publishing Workflow

1. **Research & Outline** (1-2 days)
   - Gather sources
   - Create detailed outline
   - Identify gaps

2. **Draft** (3-5 days)
   - Write initial draft
   - Add examples and details
   - Rough formatting

3. **Internal Review** (1-2 days)
   - Check accuracy
   - Verify sources
   - Improve clarity

4. **SEO Optimization** (1 day)
   - Add keywords naturally
   - Create meta description
   - Add internal links
   - Improve headings

5. **Final Proofread** (1 day)
   - Grammar and spelling
   - Formatting consistency
   - Image alt text
   - Link verification

6. **Publish & Promote** (1 day)
   - Set publish date
   - Social media announcement
   - Email notification
   - Internal linking updates
